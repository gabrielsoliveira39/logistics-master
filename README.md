**Logistics Master** é um simulador logístico interativo e educacional, desenvolvido para desafiar e aprimorar habilidades de tomada de decisão estratégica em toda a cadeia de suprimentos (Supply Chain). Criado com foco no perfil instrucional do SENAI Ourinhos, o jogo simula cenários reais enfrentados por Gerentes de Operações Logísticas em Centros de Distribuição.

---

## 🇧🇷 Sobre o Projeto

O simulador coloca o jogador no papel de um Gerente de Operações Logísticas responsável por restaurar a eficiência de um Centro de Distribuição em crise. Durante 15 rodadas, o jogador deve tomar decisões sobre Curva ABC, OTIF, roteirização, WMS, e gestão de frotas, além de enfrentar oscilações aleatórias de mercado a cada 3 rodadas.

## 🇺🇸 About the Project

**Logistics Master** is an interactive and educational logistics simulator designed to challenge and improve strategic decision-making skills across the supply chain. Created with a focus on professional training, it simulates real-world scenarios faced by Logistics Operations Managers in Distribution Centers.

### 🛠️ Development Process
This project was fully conceived and developed using the **Google Antigravity** platform, applying advanced prompt engineering and agentic AI.

- **AI Agents:** Antigravity Agent (Senior Front-End & UI/UX Expert)
- **Skill Packs:** Web Application Development Stack, SENAI Pedagogical Best Practices

---

## 🎮 Funcionalidades / Features

* **Visual de Torre de Controle:** Interface moderna inspirada em painéis de BI, dashboards ERP/WMS corporativos e sistemas de controle em tempo real.
* **15 Desafios Técnicos:** Questões com alto teor educativo baseadas em problemas práticos do dia a dia da logística.
* **10 Eventos Dinâmicos:** Oscilações de mercado que simulam o impacto de greves, alta de combustíveis e Black Friday na operação.
* **Sistema de Desempenho Dinâmico:** Feedback visual constante do seu status operacional (desde "Estado Crítico" a "Elite Operacional").
* **Responsividade Completa:** Otimizado para desktop e dispositivos móveis.
* **Zero Dependências:** Código puro, rápido e limpo em um único arquivo de distribuição.

## 🚀 Como Executar / How to Run

Como o projeto é construído estritamente com tecnologias web nativas, não há necessidade de instalação de pacotes:

1. Clone o repositório ou baixe o arquivo `index.html`.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Para publicar, basta hospedar o arquivo `index.html` na raiz do seu repositório no **GitHub Pages**.
